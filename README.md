## 介绍

Forver是一套Vue组件

> Forver is a set of Vue UI components based on Bootstrap.css

## 安装
``` bash
npm install forver --save
```

## 开始
``` js
import 'forver/dist/css/forver.min.css'
import F from 'forver'
Vue.use(F)
```

> 暂不支持 ES5 引入JS
